The Basics

The standard way to set up materials in 3D modelling tools or also game engines is to use nodes as a way to create materials. You can access them easily in Blender by using the default shading workspace. The basic functionality here is that a node processes an image or texture and then passes the result on to the next node. While the following description is based on Blender, most of the stuff described can also be found in every other node editor.

There are different node types that are typically used:

  • Input Nodes like RGB nodes offering color or image nodes for texture import
  • Processing Nodes that change value of import nodes, e.g. RGB curves like a color ramp
  • Output Nodes like the material output to show final results

In general a material has multiple properties depending on the shader used. A shader determines the color of the material when light shines. There are multiple different ones available ranging from realistic ones to toon style shaders. To achieve specific materials like mirrors a specific shader might be needed. The standard Principled BSDF Shader has multiple properties like a base color, metallic, specular or roughness

You can also set up Texture nodes as Input which then again have properties on their own, e.g. the offset or color. Typically you want to use textures and have different types like normal maps or roughness maps as well that make the illusion that there is geometry even though there isn’t. This is especially useful if you think about optimization. If you can achieve a reduction of the poly count by using a texture instead this can be useful, e.g. when modelling for video games.

UV Unwrapping

To improve how your material is mapped to your object you often will need what's called UV unwrapping. Basically what this process does is the following: Like the name suggests it takes an object (or multiple objects at once) and unwraps its surface. What you then get is a 2D representation of the surface of an object. From this you can now exactly determine, which parts of your surface sit on which part of an image texture.
