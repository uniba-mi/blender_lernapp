Theory: Modifiers

Modifiers make up an important part of modelling. We will be using some of them in future sections, in fact we start in the next one. I will not be able to explain all of them at this point in time because some of them are pretty specific and will only be applied on rare occasions. Furthermore each modifier also comes along with some properties that can be changed. In general modifiers are part of one of four categories:

  • Modify (they modify the vertex groups of objects)
  • Generate (they affect the topology by adding or removing geometry)
  • Deform (change the shape of an object without changing the topology)
  • Simulate (physical simulations)

You will often have to apply multiple modifiers together to achieve the desired result. The most important thing to remember here is that the order of the modifiers matters. They should always be applied from top to bottom.

In the following lectures we will take a look at the following modifiers that already cover most of the use cases you will need:

  • Array modifier: Generate additional geometry based on adjustable properties by repeating an object, e.g. along an axis
  • Boolean modifier: Alter the shape of an object, e.g. by cutting holes into objects
  • Mirror modifier: Used to add geometry by mirroring an object across a specific point like the origin
  • Screw modifier: Rotate an object around a point in space to create a new mesh
  • Solidify modifier: Used to add thickness to a plane surface or curve
  • Subdivision surface: Add topology to smooth out an object. Details can be found here
  • Curve modifier: Most often applied together with array modifier to repeat an object along a curve
